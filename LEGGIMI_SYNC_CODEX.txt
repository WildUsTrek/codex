# Codex Projects Sync

Questa cartella e' collegata a GitHub.

## Regola semplice

Prima di lavorare:
- doppio click su `01_AGGIORNA_PROGETTO_PRIMA_DI_LAVORARE.bat`

Per aprire PERLA1:
- doppio click su `00_APRI_PERLA1.bat`

Dopo aver finito:
- doppio click su `02_SALVA_PROGETTO_SU_GITHUB.bat`

## Nota importante su Git

Gli script non richiedono per forza Git installato nel PATH di Windows.

Metodo reale usato dagli script:

1. provano prima `where git`;
2. se Windows non trova Git nel PATH, cercano il Git incluso in GitHub Desktop:

   `%LOCALAPPDATA%\GitHubDesktop\app-*\resources\app\git\cmd\git.exe`

Quindi GitHub Desktop puo' essere aperto e funzionante anche se il terminale dice che `git` non esiste.
In quel caso non bisogna reinstallare tutto: lo script deve usare il Git interno di GitHub Desktop.
